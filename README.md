# HouseVop — Landing Page (SASS) 🏠

Проект лендинга, выполненный с использованием препроцессора SASS. Фокус проекта направлен на отработку навыков работы с компонентами, переменными и сложными сетками на десктопной версии сайта.

## 🛠 Технологии

- **HTML5:** Семантическая разметка.
- **SASS:** Модульная структура стилей.
- **Flexbox:** Основной инструмент для позиционирования элементов.

---

## 🚀 Как запустить проект

Для корректной работы всех функций (особенно внешних скриптов и путей) рекомендуется использовать локальный сервер.

### Использование Live Server (рекомендуется):
1. Откройте папку проекта в **VS Code**.
2. Установите расширение **Live Server** (если еще не установлено).
3. Нажмите на кнопку **Go Live** в статус-баре (внизу справа) или кликните правой кнопкой мыши по `index.html` и выберите **"Open with Live Server"**.
4. Проект автоматически откроется в браузере по адресу `http://127.0.0.1:5500`.
